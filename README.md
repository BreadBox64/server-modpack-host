This repo functions as both file-hosting and a working repo for a custom python script I made to update a custom minecraft modpack on multiple others' computers. Executable files are in the `/dist` folder. I wouldn't really recommend running this pack, but if you want to, you have to be using a Forge 43.3.0 MC 1.19.2 install. Download dist.zip and unzip it so the structure matches the repo, and then run the updater, telling it to reinstall the pack.
